## 系統簡介

**羽球庫存共享小幫手**是一套多球隊共用的羽球庫存管理系統，提供以下核心功能：

- 📦 **庫存追蹤**：即時掌握各球種的剩餘數量
- 📝 **領取登記**：記錄每次練球消耗的桶數與領取人
- 💰 **FIFO 結算**：依先進先出原則計算球費分攤
- 🔒 **多球隊隔離**：每個球隊的資料完全獨立
- 🏸 **開團紀錄**：記錄每場開團的場租、出席名單與利潤

## 登入與登出

### 登入

1. 在登入頁面輸入**帳號**（球隊帳號名稱）與**密碼**
2. 點擊「**登入**」按鈕進入系統

> 💡 初次使用請聯絡系統管理員取得帳號密碼。

### 登出

點擊右上角 **「登出系統」** 按鈕（桌機顯示文字，手機僅顯示圖示）即可安全登出。

## 領取登記

> 用於記錄每次練球時使用的羽球桶數。

### 步驟

1. 點擊「**領取登記**」按鈕
2. **選擇球種**：從下拉選單選擇此次使用的球種
3. **輸入領取人姓名**
4. **輸入桶數**：填入此次實際使用的桶數
5. 點擊「**確認領取**」送出

### 注意事項

- 當所有球種庫存皆為 **0** 時，「領取登記」按鈕會停用，請先入庫
- 選擇的桶數不能超過該球種的現有庫存
- 領取紀錄可在主畫面的列表中點擊刪除（誤操作時使用）

## 結算試算

> 依 FIFO（先進先出）方式計算指定期間內的球費。

### 步驟

1. 點擊「**結算試算**」按鈕
2. 設定篩選條件（均為選填）：
   - **開始日期 / 結束日期**：限定計算區間
   - **球種篩選**：只計算特定球種
   - **領取人篩選**：只計算特定人員
3. 點擊「**開始計算**」查看結果

### 結果說明

| 欄位 | 說明 |
|------|------|
| 總費用 | 篩選條件內的球費合計 |
| 球種明細 | 各球種消耗桶數與費用 |
| 計算方式 | FIFO — 依入庫批次價格依序計算 |

## 庫存管理

點擊「**庫存管理**」按鈕開啟管理介面，內含四個頁籤：

### 庫存概覽

- 以表格列出所有**啟用中**球種的庫存狀態
- 顯示欄位：球種名稱、總入庫桶數、總領取桶數、**現有庫存**

### 入庫作業

> 記錄購買新羽球並加入庫存。**需輸入入庫管理密碼（預設：1111）。**

**三步驟流程：**

1. **身份驗證**：輸入入庫管理密碼
2. **填寫資訊**：
   - 選擇球種
   - 輸入入庫桶數
   - 輸入每桶單價（元）
   - 選擇購買日期
3. **確認送出**：核對資訊後點擊「確認入庫」

> 🔐 密碼驗證通過後 **5 分鐘內**不需重複輸入。

### 入庫紀錄

- 列出所有歷史入庫記錄
- 顯示：入庫日期、球種、桶數、單價、操作人

### 球種管理

- **新增球種**：輸入品牌與型號名稱
- **停用/啟用**：停用後不再出現於領取登記選單（庫存資料保留）
- 已停用的球種在清單中以灰色顯示

## 開團紀錄

> 管理球團、記錄每場開團的場地費用、出席名單與利潤試算。

點擊頁面右上角「**開團紀錄**」按鈕進入模組。

---

### 球團管理

進入後會看到球團清單，每個球團為獨立的開團紀錄空間，以 PIN 碼保護。

#### 新增球團

1. 點擊「**新增球團**」
2. 填寫：
   - **球團名稱**（如：瘋羽無懼）
   - **隊長 / 負責人**姓名
   - **PIN 碼**（進入球團時需輸入，請妥善保管）
3. 點擊「**建立球團**」

#### 進入球團

1. 點擊球團列的「**進入**」按鈕
2. 輸入 PIN 碼後點擊「**確認進入**」
3. 驗證通過後即可查看該球團的活動紀錄

> 💡 PIN 碼驗證狀態僅保留於當前分頁，關閉分頁後需重新驗證。

#### 修改球團設定

點擊球團列右側的 ⚙️ 齒輪圖示，可修改球團名稱、隊長姓名或更換 PIN 碼。

---

### 活動紀錄

進入球團後可查看所有活動列表，並新增、查看或刪除活動。

#### 新增活動

1. 點擊「**新增活動**」
2. **（選填）貼上 LINE 報名訊息**自動解析，或手動填寫
3. 填寫場地資訊：
   - 活動日期
   - 場地名稱（選填）
   - 場地數、時數、每小時場租
4. 點擊「**建立活動**」

> ⏳ 若貼上了出席名單，建立時會逐一寫入出席者資料，可能需要約 10 秒，請耐心等候。

#### LINE 訊息自動解析

在「新增活動」視窗中，將 LINE 群組的報名訊息貼入文字框，點擊「**解析訊息**」，系統會自動擷取：

- 活動日期
- 場地名稱
- 時數
- 出席名單（依訊息順序排列）

**支援的名單格式：**

```
1.晟
2.傑
3.小豬（免費）
```

- 無名字的空白編號（如 `11.`）會自動略過
- 解析後可勾選 / 取消勾選個別出席者，或手動修改姓名與費用
- 點擊「**免費**」可標記為免費出席者（不計入收費）

#### 活動列表欄位說明

| 欄位 | 說明 |
|------|------|
| 日期 | 活動日期 |
| 場地 | 場地名稱 |
| 場租 | 場地數 × 時數 × 每小時場租 |
| 球費 | 用球成本（手動或 FIFO 試算後填入） |
| 收費 | 已繳費出席者費用合計 |
| 利潤 | 收費 − 球費 − 場租（🔴 正值紅色 / 🟢 負值綠色） |
| 狀態 | ✅ 已結算 / ○ 未結算 |

> 點擊任一列可進入「活動明細」查看出席名單與詳細資訊。

#### 刪除活動

滑鼠移到活動列會出現 🗑️ 刪除按鈕（已結算的活動無法刪除）。

---

### 活動明細

點擊活動列後開啟明細視窗，可管理出席名單與進行結算。

#### 出席名單操作

| 操作 | 說明 |
|------|------|
| 點擊費用欄 | 直接編輯該出席者的費用 |
| 點擊「已繳」按鈕 | 切換繳費狀態（已繳 / 未繳） |
| 點擊「免費」按鈕 | 標記為免費，不計入收費統計 |
| 刪除按鈕 | 移除該出席者（結算後不可刪除） |
| 新增出席者 | 在底部輸入姓名與費用後點擊「新增」 |

**批次操作（右上角）：**

- **統一費用**：一次設定所有人相同費用
- **全部已繳**：一鍵標記所有人為已繳費

#### 用球成本

可在明細視窗中手動輸入用球成本，或使用 FIFO 試算功能自動計算。

#### 結算

確認出席名單與費用無誤後，點擊「**結算此活動**」標記為已結算：

- 結算後活動無法刪除，確保歷史記錄完整
- 結算後仍可查看出席名單與費用明細

---

### FIFO 用球成本試算

在活動明細中，點擊「**FIFO 試算**」區塊，自動從庫存紀錄依先進先出原則計算用球成本。

#### 操作步驟

1. 選擇**球種**
2. 輸入本場使用的**羽球數量（顆）**
   - 系統會自動換算：**12 顆 = 1 桶**
   - 顯示換算結果：`X 顆 ÷ 12 = Y.YY 桶`
3. 點擊「**套用成本**」將計算結果填入用球成本欄位

> 💡 FIFO 試算會依照入庫批次的先後順序與單價，準確計算實際球費。

## 帳號設定

點擊右上角 **「帳號設定」** 圖示（齒輪 ⚙️）：

| 設定項目 | 說明 |
|---------|------|
| 球團名稱 | 修改顯示在主畫面的球隊名稱 |
| 登入密碼 | 修改帳號的登入密碼 |
| 聯絡信箱 | 設定用於接收通知的 Email |
| 入庫管理密碼 | 修改入庫作業所需的驗證密碼 |
| 刪除帳號 | ⚠️ 危險區域，刪除後所有資料無法復原 |

### 修改入庫管理密碼流程

1. 輸入**目前密碼**驗證身份
2. 輸入**新密碼**
3. 再次輸入新密碼**確認**

## 常見問題

**Q：忘記入庫管理密碼怎麼辦？**
A：預設密碼為 `1111`。若已修改且忘記，請聯絡系統管理員。

**Q：誤刪了領取紀錄可以復原嗎？**
A：目前無法自動復原，請謹慎操作刪除功能。

**Q：停用球種後庫存資料會消失嗎？**
A：不會。停用只是隱藏該球種，所有歷史紀錄與庫存數量均保留。

**Q：結算試算的費用計算方式？**
A：採用 **FIFO（先進先出）** 原則，最早入庫的球批次優先計入成本，確保費用計算準確。

**Q：多人同時操作會有問題嗎？**
A：系統採用資料庫層級的鎖定機制（SELECT FOR UPDATE），確保領取登記不會發生庫存超賣的情況。

**Q：新增活動時 LINE 訊息解析出現奇怪的名稱（如「12.」）？**
A：這通常是因為訊息中有空白的候補名額（如 `11.` 無填名字）。系統會自動略過空白欄位；若仍出現異常名稱，可在「出席名單預覽」中手動勾選取消或修改。

**Q：開團紀錄的利潤為什麼正值是紅色、負值是綠色？**
A：遵循台灣股市慣例，紅色代表獲利、綠色代表虧損，方便直覺判讀。

**Q：球團 PIN 碼忘記了怎麼辦？**
A：請登入後進入球團清單，點擊該球團的 ⚙️ 設定更換新 PIN 碼（無須輸入舊 PIN）。

**Q：已結算的活動可以繼續編輯嗎？**
A：已結算的活動仍可查看出席名單與費用明細，但無法刪除活動，確保歷史記錄完整。
